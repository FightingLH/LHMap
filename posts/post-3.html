多页面与单页的取舍

2025-02-02

多页面博客（MPA）下每篇文章是独立 URL，分享、收藏、搜索引擎抓取都很直接；首屏只加载当前页，对弱网和低端设备友好。

多页面的好处

无需 JavaScript 即可浏览全部内容，链接即状态，刷新不丢位置。静态 HTML 部署到 CDN 或 GitHub Pages 即可，运维成本低。

何时考虑单页

若你需要复杂交互、实时数据或应用式导航，再考虑 SPA 或静态站点生成器（如 VuePress、VitePress）。对以文章为主的博客，多页面往往更简单、可维护性更好。
