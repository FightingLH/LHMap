欢迎使用 LHMap 博客

2025-02-04

这是一个依赖 GitHub 的多页面静态博客模板。整站由纯 HTML 与 CSS 构成，无需构建工具，可直接在 GitHub Pages 上运行。

特点

多页面结构：首页、归档、关于、以及独立文章页，便于 SEO 与分享单篇文章链接。样式统一、维护简单，新增文章只需复制一篇现有文章 HTML，改标题与正文即可。

下一步

将本仓库推送到你的 GitHub，在仓库设置中开启 GitHub Pages（选择分支与目录），或使用随仓库提供的 Actions 工作流自动部署。
